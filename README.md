# Web 层项目

本仓库用于维护企业知识库问答系统的 Web 用户层模块。

## 模块目标

Web 层负责提供聊天页面，完成以下 Q1 链路：

用户输入问题 -> 调用 Agent 接口 -> 流式展示回答 -> 展示引用来源。

## Q1 必须完成

- 页面能正常加载。
- 用户可以输入问题并发送。
- 支持 Enter 发送、Shift+Enter 换行。
- 后端答案能以流式方式逐步显示。
- 回答完成后展示引用来源。
- 空输入、检索为空、后端异常、网络异常都有明确提示。

## 推荐分工

| 负责人 | 负责范围 |
| --- | --- |
| 同学 A | 页面布局、输入框、消息组件、加载状态、复制回答、清空对话 |
| 同学 B | API 调用、流式响应读取、引用来源渲染、异常状态处理、Mock 数据 |

## 目录结构

```text
web层/
  docs/       协作文档、接口约定、任务表
  mock/       Mock 响应数据或本地联调数据
  src/        Web 层源代码
  tests/      测试用例或测试记录
```

## Git 协作方式

主分支 `main` 只保存稳定版本。每个同学在自己的功能分支开发：

```bash
git checkout -b feature/chat-ui
git checkout -b feature/api-stream
```

开发完成后提交并推送，再通过 Pull Request 合并到 `main`。
