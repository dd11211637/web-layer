# 手工测试用例

本文档用于记录 Web 层本地手工测试结果。每次 Demo 前至少执行 P0 和 P1 用例。

## P0 用例

| 编号 | 场景 | 操作 | 预期结果 | 结果 | 备注 |
| --- | --- | --- | --- | --- | --- |
| MT-001 | 页面加载 | 打开 Web 页面 | 页面显示聊天区、输入框和发送按钮 | 待测 |  |
| MT-002 | 正常问答 | 输入有效问题并发送 | 显示用户问题、助手答案、引用和 `trace_id` | 待测 |  |
| MT-003 | 空输入 | 输入空字符串或纯空格并发送 | 不发送请求，提示请输入有效问题 | 待测 |  |
| MT-004 | Enter 发送 | 输入问题后按 Enter | 成功发送问题 | 待测 |  |
| MT-005 | Shift + Enter | 输入内容后按 Shift + Enter | 文本换行，不发送 | 待测 |  |
| MT-006 | 无相关文档 | 使用无相关文档 Mock | 展示知识库信息不足提示 | 待测 |  |
| MT-007 | 检索异常 | 使用检索异常 Mock | 展示检索服务异常提示 | 待测 |  |
| MT-008 | 模型异常 | 使用模型异常 Mock | 展示模型服务异常提示 | 待测 |  |
| MT-009 | 网络异常 | Agent 不可用时发送请求 | 展示网络连接异常提示，页面不白屏 | 待测 |  |
| MT-010 | 引用展示 | 使用正常回答 Mock | 展示引用编号、标题、链接或 `doc_id` | 待测 |  |
| MT-011 | 引用无链接 | 使用无链接引用 Mock | 不生成空链接，展示本地文档信息 | 待测 |  |

## P1 用例

| 编号 | 场景 | 操作 | 预期结果 | 结果 | 备注 |
| --- | --- | --- | --- | --- | --- |
| MT-101 | 流式输出 | 使用流式 Mock 或 Agent SSE | 答案逐步显示，结束后 loading 消失 | 待测 |  |
| MT-102 | 流式中断 | 中断 stream 响应 | 展示生成中断提示，页面可继续使用 | 待测 |  |
| MT-103 | 复制回答 | 点击复制回答 | 剪贴板包含助手答案正文 | 待测 |  |
| MT-104 | 清空对话 | 点击清空对话 | 聊天区恢复初始状态 | 待测 |  |
| MT-105 | trace_id 展示 | 完成一次问答 | 页面展示或控制台记录 `trace_id` | 待测 |  |
