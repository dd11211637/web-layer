# Web-Agent 联调测试用例

本文档用于记录 Web 层和 Agent 层联调时必须覆盖的接口用例。

## 联调前准备

- 确认 Agent `/api/chat` 地址；
- 确认是否支持 `stream: true`；
- 确认请求体字段：`query`、`session_id`、`stream`；
- 确认响应字段：`trace_id`、`status`、`answer`、`citations`；
- 确认异常状态是否按 `docs/接口约定.md` 返回；
- 准备浏览器控制台和网络面板截图。

## 用例清单

| 编号 | 场景 | 请求 | 预期结果 | 记录 |
| --- | --- | --- | --- | --- |
| IT-001 | 正常问答 | 有效知识库相关问题 | 返回 `success`、答案、引用和 `trace_id` | 待记录 |
| IT-002 | 空输入 | 空字符串或纯空格 | 前端不发送请求，提示请输入问题 | 待记录 |
| IT-003 | 无相关文档 | 知识库范围外问题 | 返回 `no_relevant_context`，页面展示兜底提示 | 待记录 |
| IT-004 | 检索异常 | 模拟检索服务异常 | 返回 `retrieval_error`，页面不白屏 | 待记录 |
| IT-005 | 模型异常 | 模拟 LLM 服务异常 | 返回 `llm_error`，页面不死锁 | 待记录 |
| IT-006 | 网络异常 | 停止 Agent 或断开网络 | 前端映射为 `network_error` | 待记录 |
| IT-007 | 请求超时 | Agent 延迟超过前端超时设置 | 前端映射为 `timeout_error` | 待记录 |
| IT-008 | 流式成功 | `stream: true` | token 增量显示，结束后展示引用 | 待记录 |
| IT-009 | 流式中断 | 流式响应中途断开 | 前端映射为 `stream_error` | 待记录 |
| IT-010 | 引用无链接 | 返回本地文档引用 | 页面展示 `doc_id`，不生成空链接 | 待记录 |
