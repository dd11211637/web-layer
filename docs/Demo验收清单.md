# 七月 Demo 验收清单

本清单用于七月 Demo 前逐项确认 Web 层是否可演示、可联调、可兜底。

## 页面与交互

- [ ] 页面正常加载；
- [ ] 聊天区、输入框、发送按钮正常显示；
- [ ] 用户能够输入问题；
- [ ] Enter 可以发送；
- [ ] Shift + Enter 可以换行；
- [ ] 空输入和纯空格输入不会发送；
- [ ] 发送后用户消息展示在聊天区；
- [ ] loading 状态明确；
- [ ] 清空对话功能可用；
- [ ] 复制回答功能可用。

## 问答链路

- [ ] 前端可以调用 Mock Client；
- [ ] 前端可以调用 Agent `/api/chat`；
- [ ] 普通响应可以展示完整答案；
- [ ] 流式响应可以增量展示，或已确认 Q1 使用完整答案兜底；
- [ ] 响应结束后 loading 消失；
- [ ] 每次响应能记录或展示 `trace_id`；
- [ ] 页面异常后可以继续发送下一次问题。

## 引用来源

- [ ] 能展示引用编号；
- [ ] 能展示引用标题；
- [ ] 有 `source_url` 时能点击打开；
- [ ] 无 `source_url` 时展示本地文档信息；
- [ ] 引用为空时页面不报错。

## 异常状态

- [ ] 无相关文档时展示知识库信息不足提示；
- [ ] 检索异常时展示检索服务异常提示；
- [ ] 模型异常时展示模型服务异常提示；
- [ ] 网络异常时展示网络连接异常提示；
- [ ] 请求超时时展示超时提示；
- [ ] 流式中断时展示生成中断提示；
- [ ] 所有异常场景页面不白屏、不死锁。

## 文档与记录

- [ ] `docs/接口约定.md` 与 Agent 最新接口一致；
- [ ] `docs/Mock数据说明.md` 与 `mock/*.json` 一致；
- [ ] `tests/manual-test-cases.md` 已记录手工测试结果；
- [ ] `tests/integration-test-record.md` 已记录至少一次 Agent 联调结果；
- [ ] README 能让新同学理解模块定位和目录结构。
